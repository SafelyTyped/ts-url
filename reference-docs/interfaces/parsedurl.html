Interface ParsedURL

ParsedURL holds a breakdown of a URL, using terms from the WHATWG specification.

NOTE that we do support the username and password fields, but you shouldn't use them in new code. These are deprecated by RFC 3986, and may not be supported by your choice of browser AND/OR your destination.

NOTE that we do *NOT support protocol-relative URLs here, because the underlying NodeJS URL doesn't support them.
